# KPIs — Canal USSD (TELCO, 2016–2017)

## Definiciones

- **TTGL** (Time-to-Go-Live): tiempo desde reactivación del proyecto hasta habilitación efectiva del canal.
- **TTV** (Time-to-Value): tiempo desde el Go-Live hasta observación de uso significativo en el reporte de ventas.
